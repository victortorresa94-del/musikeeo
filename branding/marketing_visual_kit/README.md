# Marketing Visual Kit

Este directorio está destinado a contener los activos visuales para marketing y redes sociales, como:

*   Banners promocionales (.jpg, .png)
*   Plantillas para Instagram Stories / Reels
*   Videos promocionales
*   Capturas de pantalla para App Store / Play Store

Referirse a `Musikeeo_Brand_System.md` Sección 12 y 16 para las guías de uso.
